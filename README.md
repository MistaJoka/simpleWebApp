# simpleWebApp

Educational Next.js app that maps **frontend vs backend** pieces of a typical web stack: UI, Route Handlers (`/api/*`), middleware, env, static assets. Local dev costs $0.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Useful endpoints: `GET /api/health`, `GET /api/pieces`. Copy `.env.example` to `.env.local` if you add env vars.

## Stack

Next.js (App Router), React, TypeScript, Tailwind CSS.
